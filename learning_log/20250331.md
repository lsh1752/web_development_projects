참조

주의 사항 : vite.config.js에서
'/여러분깃허브리포지토리명/' -> / 꼭 표기하셔야합니다.
App.jsx에서 basename=/여러분깃허브리포지토리명
이어야 합니다. 제 깃허브 확인하시고 작성해주시기 바랍니다.
npm i gh-pages
npm run build   -> predeploy를 설정하셨다면 건너뛰셔도 됩니다.
npm run deploy
